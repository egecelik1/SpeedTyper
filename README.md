# SpeedTyper 

SpeedTyper is an interactive web application designed to test and improve typing speed and accuracy in real-time. It provides users with immediate feedback on their performance, including WPM (Words Per Minute) and accuracy percentages.

## Features
- **Real-time Tracking:** Watch your speed and accuracy update as you type.
- **WPM Calculation:** Professional-grade words-per-minute measurement.
- **Accuracy Feedback:** Highlights errors to help you improve precision.
- **Responsive Design:** Fully functional on both desktop and mobile browsers.

## Built With
- **HTML5:** Semantic structure for the web.
- **CSS3:** Modern styling with Flexbox and Grid.
- **JavaScript (ES6+):** Core logic, DOM manipulation, and timer functionality.

## How to Use
1. Open the live demo link.
2. Click on the input area or the 'Start' button.
3. Start typing the text displayed on the screen.
4. View your final results once the timer ends or the text is completed.
